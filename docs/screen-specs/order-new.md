# 画面: 受注登録

- **画面 ID**: `order-new`
- **ルート**: `/orders/new`
- **初期明細行数**: 18

## ヘッダ

| 項目 | 入力方式 | 備考 |
|------|---------|------|
| 契約先コード | マスタ選択（取引先） | |
| 納入先コード | マスタ選択（取引先） | |
| 納入場所 | テキスト | |
| 納期 | 日付 | 未入力時は今日+7日を初期値 |
| 内示番号 | テキスト | 横幅 span 2 |
| 備考 | テキスト | `order-new` では Enter 順に含む。`order-new-alt` では Enter 順から除外 |

## 明細グリッド

| 列名 | 型 | 編集 | 備考 |
|------|-----|------|------|
| 行 | 行番号 | × | 左固定 |
| 製品コード | コード選択（候補リスト付き） | ○ | 確定で製品名を自動入力し、数量へフォーカス移動。セル表示は「コード 製品名」の結合形式。入力値は大文字変換 |
| 製品名 | テキスト | × | 製品コードから自動入力 |
| 数量 | 数値 | ○ | |
| 単価 | 数値 | ○ | |
| 金額 | 数値（表示のみ） | × | カンマ区切りで表示 |

### 横展開サンプル（`order-new-alt`）

上記に加えて末尾に以下の列を追加する。

| 列名 | 型 | 編集 | 備考 |
|------|-----|------|------|
| 商品分類 | テキスト（表示のみ） | × | 製品コード先頭が A → 完成品、B → 部材、それ以外 → その他 |

## 値確定ルール（副作用）

- 製品コード確定 → 製品マスタから製品名を自動入力し、数量セルにフォーカス移動
- 数量 or 単価の確定 → 金額 = 数量 × 単価 を再計算

## F キー

- F1: 新規（共通デフォルト）
- F12: 保存（共通デフォルト）
- F8: モック alert 表示

## 特記事項

- 製品コードのセルエディタはオートコンプリート（候補リスト）付き。リスト表示中は Enter がグリッドに奪われないよう制御する
- 最終行の最終編集列（単価）で Enter → 新しい行を追加し、その行の製品コードにフォーカス
- `order-new-alt` はヘッダの備考が Enter チェーンから除外される以外、`order-new` と同一動作
